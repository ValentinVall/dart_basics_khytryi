# dart_basics_khytryi

## Опис проєкту
`dart_basics_khytryi` — це навчальний репозиторій, створений для практики основ мови програмування **Dart**. У ньому реалізовано приклади командного застосунку, базову структуру проєкту, а також тестові завдання. Репозиторій демонструє стандартну організацію коду в Dart-проєктах: точка входу в директорії `bin/`, бібліотечний код у `lib/`, та модульні тести у `test/`.

---

## Структура репозиторію
- **`.idea/`** — конфігураційні файли середовища розробки (IntelliJ IDEA або інші JetBrains IDE).
- **`bin/`** — містить основний файл з точкою входу (`main.dart`), який запускає застосунок.
- **`lib/`** — бібліотечний код, що може бути використаний у `bin/` або інших частинах проєкту.
- **`test/`** — приклади модульних тестів для перевірки роботи функцій та логіки.
- **`.gitignore`** — перелік файлів та директорій, які не відслідковуються Git.
- **`CHANGELOG.md`** — журнал змін, де можна відстежувати історію версій.
- **`README.md`** — базовий опис проєкту.
- **`analysis_options.yaml`** — налаштування статичного аналізу коду Dart.
- **`pubspec.yaml`** — основний файл конфігурації пакета Dart, що містить залежності та метадані.
- **`pubspec.lock`** — зафіксовані версії залежностей для стабільності проєкту.

## Інсталяція та запуск
1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/ValentinVall/dart_basics_khytryi.git
   ```
2. Перейти в директорію проєкту:
   ```bash
   cd dart_basics_khytryi
   ```
3. Встановити залежності:
   ```bash
   dart pub get
   ```
4. Запустити застосунок:
   ```bash
   dart run bin/dart_basics_khytryi.dart
   ```
